Into to Ruby on Rails - Shopping List Web App

About the workshop

Ruby on Rails is one of the most popular frameworks for developing web applications. It is used by Hulu, Groupon, LivingSocial, YellowPages.com, Square, Visa, and many others. In this class you will learn:

  • How to make a simple dynamic web app using Ruby on Rails
  • What Ruby is, what Rails is, and why use them instead of PHP, Node.JS, Python, or .NET.
  • How to learn to code – there are hacks J.
  • Where to go from here to continue learning.

Prerequisites

  • No coding experience necessary
  • Ages 18+
  • Laptop (no netbooks) - Mac OSX Lion or newer or Windows 7
  • Complete tryruby.org tutorial - Highly recommended, only takes 15 minutes.
